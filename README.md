Vanilla JS Netflix Clone Description: A lightweight, fully responsive recreation of the Netflix web interface built entirely with standard web technologies. This project demonstrates the ability to build complex, interactive UIs without relying on heavy frameworks like React or Angular.

Features:

Authentication: Realistic login screen simulation with credential validation.

Dynamic UI: JavaScript-driven content rendering for movie rows and hero banners.

Interactivity: Custom horizontal scrolling, hover scale effects, and a dynamic "More Info" modal system.

Responsive Design: Fully adaptive layout using Tailwind CSS for mobile, tablet, and desktop views.

Tech Stack: HTML5, Tailwind CSS, Vanilla JavaScript.
